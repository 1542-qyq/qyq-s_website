乳腺癌智护小程序

Vue 移动端健康护理应用

📅 2025.03 - 2025.12 👤 前端开发（产品协作） 👥 团队项目

研发视角

1. 项目概览

一句话定位

校内高分学术项目，基于 Vue 架构开发的移动端健康护理小程序，为乳腺癌患者提供康复指导和健康管理

时间周期：2025.03 - 2025.12

我的角色：前端开发（产品协作）

团队规模：团队项目

2. 技术架构

技术栈

框架：Vue 3 + Vite UI组件：Vant 状态管理：Pinia 路由：Vue Router

3. 核心功能

健康管理模块

个人健康档案、症状记录、用药提醒

康复指导模块

术后康复训练指导、饮食建议、心理疏导

社区交流模块

病友交流、专家咨询、经验分享

4. 技术亮点

响应式设计

适配多种移动端设备，提供良好的用户体验

组件化开发

模块化设计，代码复用率高，便于维护和扩展
